How to use:

  • Enter time values as hours.minutes
    separated by '+', '-', or 'x'/'*'
  • Press Enter or '=' to calculate.
  • Input method 1: The decimal represents
    the percentage of an hour.
    Ex: 6.5 = 6hrs, 30mins
    (50% of an hour)
  • Input method 2: The decimal represents
    the minutes of an hour.
    Ex: 6.3 = 6hrs, 30mins
  • Input single value to convert.
    Ex: 3.33 + Enter/'=' = 3hrs, 20mins
  • Add/Sub/Mult two or more together.
    Ex: 2.05 + 6.72 - 1.3 x 2
  • Multiply with parentheses.
    Ex: 2.5 + (5 * 1.25)
